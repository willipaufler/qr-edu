# Newtons Gravitationsgesetz - Tipp


Für die Berechnung der Anziehungskraft nutzt du das Newtonsche Gravitationsgesetz. 

* Achte strikt auf die Einheiten: Massen in $\text{kg}$, Abstände in $\text{m}$. Recherchiere im Tafelwerk nach unbekannten Werten.
* Für Aufgabenteil b): Ein Körper befindet sich im Gleichgewicht, wenn die Kraft der Erde ($F_E$) betraglich genau so groß ist wie die Kraft der Sonne ($F_S$). Die Masse des Probekörpers kürzt sich dabei heraus.
